##  День 16: Модуль "Расписание занятий"

### Выполненные задания:

**Задание 1**: Спроектирована структура модуля
- Класс `Lesson` - модель занятия
- Класс `ScheduleEngine` - логика расписания
- Класс `Storage` - хранение в SQLite

 **Задание 2**: Реализован полный CRUD
- Создание, редактирование, удаление занятий
- Валидация времени (запрет некорректного времени)
- Проверка пересечений занятий
- Сохранение в SQLite, восстановление при перезапуске

**Задание 3**: Создан пользовательский интерфейс
- Сетка недели с временными слотами
- Клики по слотам для добавления занятий
- Таймер обратного отсчёта до следующей пары (обновление каждую секунду)

 **Задание 4**: Графическое отображение нагрузки
- Статистика по дням недели с графиками
- Статистика по типам занятий
- Цветовое кодирование типов занятий

 **Задание 5**: Документация обновлена
